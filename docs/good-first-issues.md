# Good first issues

Use these as the first public contributor tickets.

## 1. Add a real README demo GIF

- **Label:** `good first issue`
- **Scope:** capture and optimize a short GIF showing create session → save auth profile → reopen from profile
- **Why it matters:** biggest immediate launch conversion win

## 2. Add Cursor MCP setup docs

- **Label:** `good first issue`
- **Scope:** add `examples/cursor-mcp-setup.md` with copy-paste config and screenshots
- **Why it matters:** broadens adoption beyond Claude Desktop

## 3. Add SSE event stream for session status

- **Label:** `enhancement`
- **Scope:** expose a lightweight stream for session lifecycle and action events
- **Why it matters:** useful for dashboards and agent UIs

## 4. Add auth profile export/import

- **Label:** `enhancement`
- **Scope:** export a named auth profile to a portable archive and reimport it on another machine
- **Why it matters:** improves self-hosted portability

## 5. Add lightweight artifact/session replay view

- **Label:** `enhancement`
- **Scope:** simple HTML or JSON viewer for actions, screenshots, and approvals
- **Why it matters:** better debugging and better demos
