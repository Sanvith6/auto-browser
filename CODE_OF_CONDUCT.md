# Code of Conduct

Auto Browser is a technical project for serious builders.

We want the repo to stay useful, direct, and respectful.

## Expected behavior

- be respectful and professional
- assume good intent before escalating
- keep feedback concrete and technical
- welcome newcomers and answer honest questions clearly
- disagree on ideas without attacking people

## Unacceptable behavior

- harassment, threats, intimidation, or dogpiling
- hateful or discriminatory language
- doxxing or sharing private information
- sexualized language or unwanted sexual attention
- spam, trolling, or deliberately derailing threads
- encouraging abuse, stealth/evasion features, or unauthorized access

## Scope

This applies to project spaces, including:

- issues
- pull requests
- commit messages
- code review threads
- discussion and chat spaces tied to the project

## Maintainer responsibilities

Maintainers may:

- edit, hide, or remove content that violates this code of conduct
- close threads that are no longer productive
- warn, mute, or ban repeat offenders

## Reporting

If you need to report a problem, contact the repository owner privately through GitHub contact methods if available.

Do not post sensitive safety or harassment reports in a public issue.

## Enforcement

Enforcement is at maintainer discretion.

The goal is simple: keep the project constructive, safe, and focused.
